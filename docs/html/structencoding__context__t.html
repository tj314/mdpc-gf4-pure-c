Detailed Description

Structure that holds the public key.

This structure holds the second part of matrix G. In general, G = (I | -(H0*H1^-1)^T) (where H=(H0|H1)) Here, second_block_G represents the second block of matrix G, but it is not transposed: second_block_G = H0*H1^-1. Transposition is not necessary, the encoding function was written to accommodate for this.
