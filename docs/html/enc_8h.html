void enc_encrypt ( gf4_array_t *  out_encrypted,
gf4_array_t *  in_message,
size_t  num_errors,
encoding_context_t *  ctx 
)

Encrypt a message.

out_encrypted and in_message must be initialized in advance with capacity at least 2*ctx->block_size and ctx->block_size respectively. out_encrypted must be a zero array. in_message is first encoded. A random error vector with hamming weight equal to num_errors is then generated. The encrypted message is then equal to encoded+error_vector.

Parameters
out_encryptedarray to store the result in
in_messagearray containing a message to encrypt
num_errorshamming weight of the error vector to be used
ctxa valid encoding context
